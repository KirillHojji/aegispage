AEGIS

Платформа для глубокой модификации ядра Telegram. Архитектура плагинов, Xposed-хуки, Python-движок.

Техническая архитектура

Aegis — это не клиент с предустановленными модами. Это полноценная среда выполнения, внедряемая в процесс Telegram.

Plugin System

Изолированные Python-модули, работающие через Chaquopy, без необходимости модификации исходного APK при каждом апдейте.

Low-level Hooks

Прямой доступ к методам ядра через Xposed-хуки. Манипуляция событиями, базой данных и сетевыми запросами на лету.
